# ADAMRIT Meeting

*2026-06-11 — [Open in Fathom](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL)*

## Meeting Purpose

[To refine the ADAMRIT demo for a non-hospital ERP client.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=804.0)

## Key Takeaways

  - [**Demo Strategy Shift:** The demo must pivot from a hospital-specific product to a general custom ERP capability. The goal is to build client trust by showing a working product, then present a tailored solution for their gas company.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3064.0)
  - [**Client-Specific Focus:** The demo must align with the client's current Odoo modules (e.g., finance, inventory, HR) and their specific requirements (e.g., GPS tank level integration), not just hospital features.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=1422.0)
  - [**Technical Architecture Defined:** The solution will use Tally for core accounting (the system of record) and MicroVista for e-invoicing. It will be hosted on OCI Cloud in the UAE, using OCI's native AI services for cost-effectiveness and data residency.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=2481.0)
  - [**Key Differentiator:** The "Agent Factory" is the core value proposition. It allows users to create custom AI agents (not chatbots) to automate ERP tasks, overcoming common user resistance to manual data entry.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=2873.0)

## Topics

### Demo Strategy & Client Context

  - [**Client:** PetroSafe and Metals, a gas company using Odoo ERP.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=1988.0)
  - [**Objective:** Position ADAMRIT as a custom ERP alternative to expensive, subscription-based standard ERPs.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=4030.0)
  - [**Demo Goal:** Build trust by showing a working product (ADAMRIT) as proof of capability, then present a tailored solution for the client's specific needs.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3079.0)
  - [**Required Shift:** The demo must focus on general ERP functions (finance, inventory, HR) and map to the client's current Odoo modules.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=1422.0)

### PetroSafe ERP: Technical Architecture & Execution

  - [**Accounting:**](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=2481.0)
      - [**System of Record:** Tally Prime, which is FD-approved in the UAE.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=2800.0)
      - [**E-Invoicing:** MicroVista.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=2481.0)
      - [**Rationale:** This hybrid approach provides a robust, compliant accounting backbone, avoiding the risk of AI-driven financial errors.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=2863.0)
  - [**Hosting & AI:**](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3760.0)
      - [**Infrastructure:** OCI Cloud in the UAE.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3791.0)
      - [**AI Services:** OCI's native AI services.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3791.0)
      - [**Rationale:** Ensures data residency (critical for Emirati clients) and offers a cost-effective alternative to self-hosting open-source LLMs.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3760.0)
  - [**Execution Plan:**](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3122.0)
      - [**Parallel Run:** The new ERP will run alongside Odoo for a 90-day stabilization period.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3157.0)
      - [**Phased Migration:** Functionality will be migrated from Odoo to the new ERP module by module.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3157.0)
      - [**Odoo Retirement:** Odoo will be fully retired after the stabilization period.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3162.0)

### Key Differentiator: The "Agent Factory"

  - [**Concept:** A platform for users to create custom AI agents that automate tasks.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3274.0)
  - [**Functionality:**](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3431.0)
      - [**Task Definition:** Users describe a task in natural language.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3431.0)
      - [**AI Analysis:** The system breaks the task into steps and generates a workflow.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3431.0)
      - [**Integration:** Connects to external tools like Slack and WhatsApp for notifications.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=3492.0)
  - [**Value Proposition:** Overcomes user resistance to manual ERP data entry by enabling them to automate their own workflows.](https://fathom.video/share/FqjpzXoTsiexW2HNq2zPE4FXZsPR6NeL?tab=summary&timestamp=2925.0)
