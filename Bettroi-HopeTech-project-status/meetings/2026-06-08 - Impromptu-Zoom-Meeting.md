### New Project Management Workflow

  - [**Ownership:** One team lead per project is responsible for requirements and on-time delivery.](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=74.0)
  - [**Slack Thread Structure:**](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2308.0)
      - [**Format:** `[Icon] (Domain Name) Owner Name`](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2308.0)
      - [**Example:** `(Pulse.org) Tarut`](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2357.0)
      - [**Rationale:** The domain name provides a durable, unambiguous project identifier.](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2345.0)
  - [**Google Doc Structure:**](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2510.0)
      - [**Format:** One single Google Doc per project, with all content pasted directly into it.](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2120.0)
      - [**Rationale:** Creates a unified, searchable document, avoiding broken links and versioning issues.](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2120.0)
      - [**Required Headings:**](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2510.0)
          - [`Pre-order documents`](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2510.0)
          - [`Post-order documents`](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2510.0)
          - [`Client Share documents`](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2510.0)
          - [`Send to client job documents`](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2510.0)

### QA & Git Workflow

  - [**Git Workflow:** Push to feature branches → merge to `main`.](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2643.0)
  - [**QA Process:**](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2670.0)
    1.  [**Self-Testing:** Developer tests new feature and confirms no regressions.](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2670.0)
    2.  [**Peer Review:** A second developer performs the same two tests.](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2670.0)
  - [**Bug Prioritization (P-Levels):**](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2777.0)
      - [`P1`: Critical (client cannot work)](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2777.0)
      - [`P2`: High (major feature broken)](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2777.0)
      - [`P3`: Minor bug](https://fathom.video/share/aPppzat_iHe5ihPrux_gZfxntnfK-AtR?tab=summary&timestamp=2777.0)
