  - [**Problem:** A developer asked a client a basic question about removing WordPress, revealing a team-wide lack of project clarity.](https://fathom.video/share/zsQmfcsimL3EZcn946jox57wxzfzKrsh?tab=summary&timestamp=25.0)

      - [Haritha was not included in project plan reviews before client demos.](https://fathom.video/share/zsQmfcsimL3EZcn946jox57wxzfzKrsh?tab=summary&timestamp=558.0)

  - [**Solution:** Implement a new process to ensure team alignment and quality.](https://fathom.video/share/zsQmfcsimL3EZcn946jox57wxzfzKrsh?tab=summary&timestamp=300.0)

      - [**Internal QA:** Perform full functional testing after any deployment or server change, *before* client UAT.](https://fathom.video/share/zsQmfcsimL3EZcn946jox57wxzfzKrsh?tab=summary&timestamp=1302.0)
      - [**Project Briefs:** Create concise documents for developers to reference, reducing reliance on memory and recordings.](https://fathom.video/share/zsQmfcsimL3EZcn946jox57wxzfzKrsh?tab=summary&timestamp=300.0)
