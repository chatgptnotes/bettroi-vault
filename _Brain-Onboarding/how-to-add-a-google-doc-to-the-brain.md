# How to Add a Google Doc to the Brain

**Hope Hospital · Bettroi — Second Brain**

Follow these 3 steps and the Brain will read your document automatically. After that, you (and the team) can ask `@brain` questions about it in Slack.

---

## The 3 steps

**1. Open the doc → click the blue _Share_ button** (top-right).

**2. Under _General access_, set it to "Anyone with the link" → _Viewer_.**
   - Change "Restricted" to **Anyone with the link**
   - Set the role to **Viewer** — **not** Editor
   - _(Viewer lets the Brain read it. Editor would let anyone with the link change it — don't.)_

**3. Click _Copy link_, then paste that link into the project's Slack channel or thread** (any channel where `@brain` is a member).

The Brain ingests it within a few minutes. Done.

---

## ⚠️ Do NOT share these

"Anyone with the link" means **anyone who has the link can open it.** Never share a doc that contains:

- **Passwords, API keys, or logins**
- **Patient information (PHI)**
- **Bank, salary, or financial details**

If a doc has secrets mixed in, make a clean copy without them and share that instead.

---

## Quick tips

- **Post the link in the right project's channel/thread** — the Brain tags the doc to that project, so answers stay organised.
- **Viewer is enough.** You never need to give Editor access for the Brain.
- **One link per doc** is plenty — you don't need to re-paste it later; updates are picked up.
