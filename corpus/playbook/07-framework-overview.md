# Be AI-First Playbook — Framework Overview

> Source: https://docs.google.com/document/d/1QXnapEEbR6huinDwa-8ho2BfN1lcRCCvhqkaAxI74mM
> Surfaced publicly at https://2men.co/framework so prospects can read the methodology before paying.
> The 10-step interactive playbook (`04-step-by-step.md`) is the productized execution of this framework.

The Be AI-First Framework is a methodology for transforming a business into an AI-native operating system where AI handles **60–80% of workflows**. Four phases, each ending in a verifiable deliverable.

## The 4-Step Playbook

### Step 1 — Learn (Weeks 1–2)
Focus on building personal conviction by using tools like Claude Code or Claude Co-Work daily for 1–2 weeks. You must understand the tool's capabilities firsthand to avoid mediocre system design.

### Step 2 — Wire (Weeks 3–5)
Build your "business brain" by structuring all company knowledge — processes, team structures, pricing, and client data — into a format (like Markdown / Obsidian) that is queryable and legible to AI. Connect live data sources like Slack to keep the system current.

### Step 3 — Automate (Weeks 6–10)
Map out every department (marketing, sales, delivery, ops) and create AI agents with specific skills. Implement closed loops (self-improving feedback) and test harnesses — checklists that define "good enough" — so the AI can iterate on its own output before it reaches you.

### Step 4 — Scale (Week 11+)
Apply "token maxing" economics. Instead of scaling via headcount, you scale by increasing AI output. Use the time saved to multiply your efforts, allowing one person to perform the work of an entire team.

## Core Principles

- **Closed Loops** — unlike open-loop systems where tasks are executed once, closed-loop systems continuously monitor output, analyze results, and feed insights back to get smarter over time.
- **Business Brain** — all company knowledge structured in a format AI agents can query, cite, and act on. Markdown with frontmatter; not PDFs.
- **Test Harnesses** — predefined criteria the AI uses to self-check its output before a human reviews it. The agent's "definition of done".
- **New Org Structure** — replace traditional middle management with an intelligence layer; humans focus on judgment (IC / DRI / AI Founder).
- **Token Maxing** — scale output by increasing tokens-per-dollar and tokens-per-task, not by adding headcount. Tracked via the AI Leverage Ratio.
